## Install 📥

```bash
npm i -g customizable-gitmoji-cli

yarn global add customizable-gitmoji-cli
```

(If you installed _original_ gitmoji-cli)

```bash
npm uninstall -g gitmoji-cli

yarn global remove gitmoji-cli
```

## Usage 🧰

```bash
gitmoji init
```
